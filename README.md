# 高精度自動回放與錄製工具 (MacroApp)

這是一個具備圖形化介面（GUI）的鍵盤滑鼠巨集工具，提供高精度的按鍵錄製與回放功能。具備強大的「CPS 過濾限制」與「F8 緊急中斷」機制，確保回放過程的安全與穩定。

## 🌟 核心功能

1. **圖形化介面 (GUI)**：直觀的視窗操作，可輕鬆切換錄製與回放模式。
2. **高精度回放**：採用 `perf_counter` 與微空轉（Busy-wait）補償技術，解決 Python 在 Windows 下 `time.sleep` 精度不足的問題。
3. **點擊過濾 (CPS Limit)**：可自訂每秒最大點擊上限，智慧過濾超過設定閾值的過快按鍵事件。
4. **全域緊急中斷**：執行錄製或回放時，隨時按下 **`F8`** 鍵即可安全且立即地中斷任務，並會自動釋放按住的按鍵，避免卡鍵。
5. **設定自動記憶**：自動建立並讀取 `setting.json`，儲存您上次的操作參數。
