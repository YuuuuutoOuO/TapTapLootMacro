# 高精度自動化工具 (MacroApp) V3.2 智慧防護版

這是一個專為 Windows 環境設計的高精度按鍵錄製與自動化回放工具。具備雙執行緒流暢介面、極低延遲的回放補償技術、智慧防誤觸機制，以及一鍵生成循環腳本的功能。

## 🌟 核心特色 (Features)

* **🚀 高精度引擎 (High Precision)**
  * 捨棄傳統的 `time.sleep()`，採用 `time.perf_counter()` 搭配微空轉 (Busy-wait) 技術，達成微秒級的精確回放，解決連續點擊節奏失常的問題。
* **🛡️ 智慧防護機制 (Smart Anti-Mistouch)**
  * **提早鎖定 (Early-Lock)**：倒數期間只要切換至目標視窗即刻鎖定。
  * **失焦暫停 (Focus-Lost Stop)**：執行期間若滑鼠誤觸其他視窗或跳出通知，系統會以毫秒級速度自動緊急煞車，防止巨集在桌面或其他軟體失控狂點。
* **⚙️ 雙執行緒架構 (Dual-Thread UI)**
  * UI 介面與底層按鍵邏輯徹底分離，執行期間主程式絕不卡死，確保「停止按鈕」與「狀態更新」隨時保持最高反應速度。
* **🤖 自動生成腳本 (Auto-Generator)**
  * 無需手動錄製！輸入目標的「每秒點擊次數 (CPS)」，系統會利用亂數動態切割時間，自動生成極度擬真的 `0~9` 數字循環點擊腳本。
* **🚨 雙重緊急中斷**
  * 支援全域熱鍵 **`F9`**，或是點擊 UI 上的實體 **⏹ 停止執行** 按鈕，皆可瞬間終止所有動作並釋放卡鍵。
* **📁 綠色便攜 (Portable)**
  * 打包成 `.exe` 後，所有的錄製檔 (`record.json`) 與設定檔 (`setting.json`) 皆會自動存放於執行檔同目錄下，隨身碟插了就走，不會殘留於系統暫存區。
